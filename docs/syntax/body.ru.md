# Тело

Тело - это участок кода, список строк которые находяться под конкретным обьектом.
Перед определением тела используеться символ `:` **всегда**.

Тела деляться на полные и простые (одно-линейные).

Пример кода с полным телом:

```ql
function main(): {
    Console.println("Hello, World!");
    return;
}
```

Простое тело используеться когда код являеться одно-строчным:

```ql
if Console.readln() == "0":
    Console.println("You printed '0'.");
```

> ВАЖНО: Простое тело не может быть использовано при определения [класса](class.md)!

## Limitations

В теле в некоторых случаях не может быть прописаны: [функции](function.md), [классы](class.md) и модификатор приватности поля (`private`).

Пример ***неправильного кода*** в [классе](class.md):

```ql
class MyClass: {
    // Выдаст ошибку, посколько прописаны в теле класса.
    class OtherClass: {

    }
}
```

Пример ***неправильного кода*** в [функции](class.md):

```ql
function main: {
    // Выдаст ошибку, посколько прописаны в теле функции.
    class OtherClass: {

    }

    // Выдаст ошибку, посколько прописаны в теле функции.
    function otherFunction(): {

    }

    // Выдаст ошибку, посколько модификатор `private` прописан в теле функции.
    private const variable = 0;
}
```
