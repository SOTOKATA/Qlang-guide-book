# "Привет, мир!" программа

## Code

Код для написания "Hello, World!" в консоль:

```ql
include "$lib/base"

function main(): {
    Console.println("Hello, World!");
}
```

## Разбор

Первое и самое важное - это `include`.
Это комманда по добавлению одного файла в другой, прицип похож на C++ метод, но не равен.

В данном варианте мы добавляем стандартную библиотеку из компилятора "$lib/base".

Смотрите также страницу: [Что такое `include`?](include.md).

Второе - точка входа в программу, она аналогична C++, просто создайте функцию с названием `main`, вам не обязательно принимать аргументы.

Смотрите страницу: [Функции](function.md)

И сама команда по написанию в консоль - это `Console.println("Hello, World!")`.
В этой строке происходит вызов класса `Console` из `$lib/base`, потом в классе мы вызываем функцию `println()` и передаем аргумент типа `String`, `"Hello, World!"`.

Смотрите страницу: [Функции](function.md)
