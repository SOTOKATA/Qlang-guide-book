# Функция

> Перед продчениям данной страницы рекумендуеться прочитать про [тела](body.md).

Функции являються блоками кода с именами, которые можна вызывать.
Функции могут возращять нечего, или разные значения и типы.
Они похожы на функции из javascript.

Пример функции которая являеться точкой входа в программу:

```ql
include "$lib/base"

function main(const args): {
    // Body is here.
}
```

Функция называеться `main` и принимет аргумент `args`.

Если необходимо, функция может принять типизированый аргумент:

```ql
function print(const<String> toPrint): {

}
```

В данном случае функция `print(const<String> toPrint)` принимает аргументы лишь типа `String`. При передаче другого типа будет вызвана ошибка о не совместимосте типов.

Функции имеют приватность, если они прописаны в классе:

```ql
class MyClass: {
    private function privateFunction(): {

    }
}
```

Данная функция будет доступна **только** в классе `MyClass`.

Все функции обычно публичные, если не прописан keyword `private`.

## Вызов

Чтобы вызвать функции достаточтно просто написать имя и агрументы:

```ql
functionExample();
// Или если функция имеет аргументы:
functionExampleWithArgs("Hello, World!");
```

## Указательная функция

Данная функция являеться стрелочной (или же лямбда) функцией, и определяеться в переменную:

```ql
const variable = function () => {
    // Function body.
};
```

Для вызова такой функции достаточно просто написать имя переменной и добавить скобки:

```ql
const variable = function () => {
    // Function body.
};

// Call
variable();
```
