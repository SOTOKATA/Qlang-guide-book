# Подключить

`include` - это способ добавить один файл в другой. Фактически код будет скопирован из одного файла в другой.

Чтобы добавить файл следует прописать данный код:

```ql
include "path/to/file.ql"
```

Путь к файлу являеться относительным того файла, в который будет добавлен.

Если нужно прописать добавление всех файлов в папке следует просто прописать путь к папке. Компилятор сам добавит все файлы из папки с разширением `.ql`. Компилятор не будет добавлять файлы, которые находяться в суб-папке, то-есть:

```dir
directory/
    file.ql
    other_directory/
        console_op.ql
```

Строка `include "directory"` добавит файл `file.ql`, но не добавит файл `console_op.ql` так как он находиться в суб-папке.

## Limitations

Так как это указатель компилятору, строка должна быть обязательно вида: `include "путь"`.

```ql
// Правельный код:
include "directory" 

// Код, который выдаст ошибку:
include "path.ql" // comment
/* comment */include "path.ql"
include file
include 'file_path.ql'
    include          "haha.ql";
include "code.ql";
```
