# Класс

> Перед продчениям данной страницы рекумендуеться прочитать про [тела](body.md).

Класс - это структура для хранения функций и переменных.

Класс в Qlang - это пройстой способ упростить код через ООП.
Классы по функционалу аналогичны другим языкам програмирования.

## Создание класса

Создания класса происходит через конструкцию `class [class_name]:`, класс обязан иметь полное тело.
Пример создания класса `MyClass`:

```ql
class MyClass: {
    function new(): {

    }
}
```

Функция `new()` являеться автоматической функцией для создания нового класса, она не являеться обязательной и по стандарту являеться пустой.

> ВАЖНО: Данная функция **всегда** возращяет класс, в ней недопустим keyword `return`!

## Использования класса

Любой класс может быть статическим и динамическим:

```ql
class MyClass: {
    function new(): {

    }

    // Функция для написания "Hello, World!" в консоль
    function printHelloWorld(): {
        Console.println("Hello, World!");
    }
}

function main(): {
    // Статическое использование
    MyClass.printHelloWorld();

    const myClass = MyClass.new();

    // Динамическое использование
    myClass.printHelloWorld();
}
```

Для создания копии класса следует использовать функцию `new()`:

(В данном случае создаться новый String)

```ql
String.new("Hello, World!");
```

## Наследовательность

Один класс может наследовать другой.
Наследуемость работает через добавление переменных и функций из наследуемого класса в основной:

```ql
class OtherClass: {
    const variable = 1;

    function fn(): {

    }
}

class MyClass extends OtherClass: {
    // В этот класс будут добавлены функция `fn` и переменная `variable` из класса `OtherClass`
} 
```

Чтобы перезаписать функцию следует просто заново обьявить ее.
Следует внимательно следить за тем, какие функции перезаписаны, чтобы не запутаться в коде.

## Internal функции

Internal функции - это функции которые есть в любом классе, но не являються обязательными для перезаписи.

Список всех Internal функций:

- `function ___create_from___(const obj)` - функция для создания класса из другого типа (Возвращяет: класс, в котором прописана функция)
- `function ___operator_plus___(const obj1, const obj2)` - функция для обработки оператора `+` (Принимает два класса, возращяет такой же класс)
- `function ___operator_minus___(const obj1, const obj2)` - функция для обработки оператора `-` (Принимает два класса, возращяет такой же класс)
- `function ___operator_star___(const obj1, const obj2)` - функция для обработки оператора `*` (Принимает два класса, возращяет такой же класс)
- `function ___operator_slash___(const obj1, const obj2)`  - функция для обработки оператора `/` (Принимает два класса, возращяет такой же класс)
- `function ___operator_equal_equal___(const obj1, const obj2)` - функция для обработки оператора `==` (Принимает два класса, возращяет bool)
- `___operator_not_equal___(const obj1, const obj2)` - функция для обработки оператора `!=` (Принимает два класса, возращяет bool)
- `function ___operator_greater_equal___(const obj1, const obj2)` - функция для обработки оператора `>=` (Принимает два класса, возращяет bool)
- `function ___operator_less_equal___(const obj1, const obj2)` - функция для обработки оператора `<=` (Принимает два класса, возращяет bool)
- `function ___operator_greater___(const obj1, const obj2)` - функция для обработки оператора `>` (Принимает два класса, возращяет bool)
- `function ___operator_less___(const obj1, const obj2)` - функция для обработки оператора `<` (Принимает два класса, возращяет bool)

В осномном Internal функции нужны для проведения операций над классами.

## Указательные классы

Данный класс являеться упрощенным (не имеет статического тела, и имеет в себе только переменные) и определяеться в переменную:

```ql
const variable = {
    const variable = 1,
    const functionExample = function () => {

    }
};
```

Для вызова переменной из класса достаточно просто написать имя переменной и добавить название другой переменной:

```ql
// Creation
const object = {
    const variable = 1,
    const functionExample = function () => {

    }
};

// Call
object.variable;
object.functionExample();
```
