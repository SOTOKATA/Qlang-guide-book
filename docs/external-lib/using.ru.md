# Использование внешних библиотек в Qlang

Каждая внешняя библиотека - это папка с `.dll` файлами.

Для добавления внешней C# библиотеки следует использовать `native`.

`native` - это ключевое слово, которое аналогично `include`, но добавляет внешние библиотеки.

Пример использования:

```ql
native "path/to/dll/folder"
```

`native` принимает только папки и включает в проект **все** файлы типа `.dll` в ней (не учитывая суб-папки).

После этого вы можете использовать специальную функцию `_native` для вызова функций из C# библиотек.

Функция принимает неограниченое количество аргументов, но первый - это всегда путь к функции.
Путь состоит из `Namespace`, `Class`, `FunctionName`: "namespace.class.function". Также функция `_native` может возращять значения, если это прописано в реализации C# функции.

Пример вызова функции `write`, которая описана в статье: [Создание библиотеки](creation.ru.md):

```ql
native "sotokataLib"

function main(): {
    _native("sotokata.console.write", "Hello, World!");
}
```
