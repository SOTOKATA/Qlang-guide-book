# Установка

Инструкция:

1. Скачайте последнюю версию Qlang из [Github репозитория](https://github.com/SOTOKATA/Qlang) (для вашей ОС).
2. Разархивируйте `.zip` архив.
3. Перенесите в удобное для вас место папку с компмлятором.
4. (Не обязательно, но **желательно**) Добавте в список комманд:
    42. Для Windows: добавте в PATH путь к компилятору.
    43. Для MacOS: ...
    44. Для Linux: ...
5. Похлопайте в руки и порадуйтесь проделаной работе!
