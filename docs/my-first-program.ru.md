# My first program

Язык имеет систему проектов, то-есть любая программа изначально проект а не просто файлы.
Основная статья по проектам находиться [тут](project.md).

## Создание программы

Для создания проекта следует написать команду в консоле:

```command
ql new <project-name>
```

Вместо `<project-name>` следует написать название вашого проекта

> ВАЖНО: Потдерживаються лишь англиские символы, точки, подчеркивание и дефисы!

После выполения команды создастся папка с именем проекта. В ней будут находится несколько файлов, нас интересует лишь `main.ql`
В нем уже будет написан пример программы, которая выведет "Hello, World!" в консоль:

```ql
include "$lib/base"

function main(): {
    Console.println("Hello, World!");
}
```

Для полного разбора данного примера смотрите статью: ["Hello, World!" программа](syntax/hello-program.md).

Чтобы запустить программу следует в **директории проекта** прописать комманду:

```command
ql run
```

Данная команда запустить компиляцию и выполнение кода.

Поздравляю, вы написали свою первую программу! Чтобы лучше разобраться в коде прочитайте: [Синтаксис языка](syntax/introduction.md).
