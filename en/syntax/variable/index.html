Variable

Перед продчениям данной страницы рекумендуеться прочитать про тела.

Переменные являються контейнерами для значений. В Qlang переменные не строго типизированы, но имеют мутабельность аналогичную javascript.

Пример создания переменной, которая будет равна значению 42:

const variable = 42;

Переменную variable нельзя будет изменить в будущем, то-есть она не имутабельна.

Если переменную нужно менять в дальнейшем, то следует использовать keyword let при создании:

let variable = 42;

variable = 32;

Переменная будет сначала равна 42, но потом изменена на 32.

Переменные имеют приватность, если они прописаны в классе:

class MyClass: {
    private const var = 1;
}

Данная переменная будет доступна только в классе MyClass.

Все переменные обычно публичные, если не прописан keyword private.
