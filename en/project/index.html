Project

Язык как и C# использует структуру проекта. Проект - это просто удобный способ струтурировать файлы, зависимости и настройки программы.

Создание

Чтобы создать новый проект следует прописать команду:

ql new <project_name>

После выполнения будет создана папка с названием проекта, если она уже есть, то будет перезаписана.

Каждый проект состоит из таких файлов:

  • main.ql
  • project.settings.json
  • compile.settings.json

Эти файлы являються обязательными, и при удалении или нарушении целосности приведут к поломке всего проекта.

В файлах с суфиксом settings храняться настройки проекта.

Файл main.ql будет использован при начале компиляции и должен содержать функцию main.

Настройка

Чтобы настроить settings файлы нужно прописать следущую команду:

ql set [param] [value]

set - команда, которая означает изменение параметра

[param] - параметр, который нужно поменять.

[value] - новое значение.

Если нужно получить информацию про параметр, следует прописать команду:

ql get [param]

get - команда, для получение информации про параметр.

[param] - название параметра.

Удаление

Просто удалите папку с проектом.
