My first program

MANDATORY: Before doing this, complete the section on installing the compiler!

To create your first program, you should create a project.

To create a project, use the new command:

ql new [project_name]

After that, a folder named [project_name] will be created with the program “Hello, World!” already written inside. To start the project (and compile it), use the command:

ql run

The compiler will automatically find the main.ql file, then create an executable file with the program and run it. The program usually runs in console mode.
