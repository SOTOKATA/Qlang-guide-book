Project

The programming language uses a project structure for every program. This means that in order to create the final program, you need to create a project.

Project creation

To create a project, use the new command:

ql new [project_name]

A folder named [project_name] will be created.

Each project contains the following files:

project_name/
main.ql
project.settings.json
compile.settings.json

main.ql - a file that will be processed by the compiler.

project.settings.json - project parameters (name, as an example).

compile.settings.json - compilation parameters (using debug mode as an example).

These files are mandatory, and without them the project will be broken.

Project configuration

Project parameters can be configured using the set command:

ql set [param] [value]

[param] - the parameter that should be changed.

[value] - the new value of the parameter.

To obtain information about the current value of a parameter, use the get command:

ql get [param]

[param] - the parameter for which information should be obtained.

Deleting a project

Just delete the project folder.
