Introduction

Installation instructions can be found here.

Qlang (Quick Language) is a programming language created in 2025 as a final project by a student. The language is not competitive with other languages and was created solely as an example.

How does it work?

Qlang works as a command translator to C#, i.e. the program first "translates" the code into C# and only then into a binary file. When compiling into .exe (or other executable files), the language simply compiles the C# program with your program class.

What can I use it for?

For anything, you can even write your own C# libraries for the language, which means you can use it in the same way as C#. The language also has a built-in library for working with Raylib (via Raylib_cs). Instructions for use can be found here.

Should I use it?

If it's for serious use, then no. The language isn't that advanced, and it's easier to just use C#.
