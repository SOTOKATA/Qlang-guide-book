Syntax

The syntax of the language is similar to JavaScript, but also has elements from C++ and C#.

An example of a program that will display Hello, World! in the console:

// Adding a standard class package (in our case, the `Console` class)
// This file must be added because it contains the core functionality of the language.
include "$lib/base"

// Creating the `main` function, which will be called when the program starts
function main(): {
    // Call the `println` function from the `Console` class and pass the argument `“Hello, World!”`
    // To end the line, use `;`
    Console.println("Hello, World!");
}

Class

Creation

To create a class, you need to write the keyword class and the class name:

class MyClass: {
    // Function that will be called when creating a new copy of the class.
    // JavaScript equivalent: constructor function
    function new(): {

    }
}

Inheritance

A class can also inherit other classes:

class MyClass extends OtherClass: {
    // Class body.
}

Functions and variables from the OtherClass class will be added to MyClass.

Creating a copy

Creating a new copy:

const myClassInstance = MyClass.new();

Functions

Definition

The definition can be made in class or in a file.

When declared in a class, the private modifier is also available. When added, the function can be called only in the parent class.

To declare a function, use the keyword function, the function name, and the parameters in parentheses (if any):

function main(): {
    // Function body.
}

function functionExample(const<String> var): {
    // Body recived var (must be `String` type).
}

class MyClass: {
    // Acess only from `MyClass` (`MyClass.classFunction()`) or in `MyClass` (just: `classFunction()` or with keyword `this`)
    function classFunction(): {

    }

    // Access only in `MyClass`
    private function privateClassFunction(): {

    }
}

Function call

To call a function, write its name and parameters in parentheses (if any):

function main(): {
    // Call function.
    functionExample("Hello, World!");
}

function functionExample(const<String> var): {
    // Body recived var (must be `String` type).
}

Variables

Definition

Declarations occur in classes, functions, blocks, and simply in a file. When declaring, you should specify the mutability of the function (const - immutable, let - mutable, as in JavaScript).

When declared in a class, the private modifier is also available. When added, the variable can be called or changed only in the parent class.

// Access available from anywhere
// This method is not recommended because the variable can also be obtained in other files.
const global_var = 1;

class MyClass: {
    // Access only from or in `MyClass`
    const class_var = 1;

    // Access only in `MyClass`
    private const private_class_var = 1;
}

function main(): {
    // Access only in function `main`
    let name = "Volodymyr";
}

Get and change

To obtain the value of a variable, simply write the path to it. To change the value, you need to write the path to it and then write equal and the value.

class MyClass: {
    let var = 10;

    function myFunction() {
        // Change the value of `var`
        var = 20;
    }
}

function main() {
    // Now value of `var` is changed
    MyClass.var = 30;

    const myClassInstance = MyClass.new();

    // Now value of `var` is changed in the instance `myClassInstance`
    myClassInstance.var = 40;
}
